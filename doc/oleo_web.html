Oleo is GNU's spreadsheet.

Oleo has not been actively maintained in quite a while. This has changed !

The current release of oleo is 1.99.5 .

The previous release (oleo-1.6.tar.gz) is still available but it's very old. We're working on a release with lots of bugfixes and some exciting new features to arrive shortly.

Release 1.99.5 has been made available on the regular FTP site, all the GNU mirrors should pick it up. It should be stable enough for many purposes, it's certainly more useful than release 1.6.

Intermediate versions are available at ftp://alpha.gnu.org/pub/gnu/oleo . The latest and greatest release currently available is oleo-1.99.5 (available from July 1999).

Changes w.r.t. oleo 1.6 :

  • Motif/LessTif interface, based on the XbaeMatrix widget
  • GNU plotutils graphics (only reachable from Motif/LessTif interface)
  • more portable build through autoconf and automake
  • internationalised (i18n)
  • (somewhat) updated documentation
  • ANSI C code (removed K&R stuff)
  • filter mode (read file from stdin, save to stdout)
  • better SYLK support
  • improved printing (PostScript and PCL)
  • database access (MySQL and DBF files)
  • SciPlot graphics (Motif/LessTif only)
  • cleaned up internals
  • runs in character mode as well as in a GUI

This mailing list concern oleo :

  • bug-oleo@gnu.org for reporting (and reading about) bugs in oleo.
    send mail to bug-oleo@gnu.org
    subscribe to bug-oleo (be sure to put the word subscribe in your subject line)

The GNU mailing lists maintain archives of the messages sent to it. The archives can be queried by E-mails whose subject starts with the word archive. By sending a message with subject archive help to bug-oleo-request@gnu.org you will ask the server to send you a message with instructions for working with the archive server.

We recommend that you also send a message with the subject archive ls latest as "latest" is the name of a directory.

Please note that LessTif is a freely available replacement for OSF/Motif. LessTif is licensed by the GNU LGPL and can be obtained from www.lesstif.org.
