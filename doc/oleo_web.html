Oleo is GNU's spreadsheet.

Oleo has not been actively maintained in quite a while. This has changed !

The current release of oleo is still 1.6, but we're working on an intermediate release with lots of bugfixes and some exciting new features to arrive shortly.

Those who want to have a look at intermediate versions, take a look at ftp://alpha.gnu.org/pub/gnu/oleo.

Changes w.r.t. oleo 1.6 :

  • for better portability, the build is done with autoconf and automake
  • oleo is now internationalised
  • updated documentation
  • ANSI C code (removed K&R stuff)

Stuff that we're working on :

  • GUI interface (someone's working on a Motif/LessTif interface, someone else is working on a GTK interface)
  • GNU plotutils graphics
  • SciPlot graphics (Motif only)

Oleo features :

  • reads and writes several file formats : SYLK, SC, ... FIX ME
  • spreadsheets of up to 65536 rows and columns
  • supports (simple) graphics
  • runs in character mode as well as in a GUI

There are two mailing lists about oleo :

  • bug-oleo@gnu.org for reporting (and reading about) bugs in oleo.
    send mail to bug-oleo@gnu.org
    subscribe to bug-oleo (be sure to put the word subscribe in your subject line)
  • help-oleo@gnu.org for asking and providing help in how to use oleo.
    send mail to help-oleo@gnu.org
    subscribe to help-oleo (be sure to put the word unsubscribe in your subject line)

The mailing lists maintain archives of the messages sent to it. The archives can be queried by E-mails whose subject starts with the word archive. By sending a message with subject archive help to help-oleo-request@gnu.org you will ask the server to send you a message with instructions for working with the archive server.

We recommend that you also send a message with the subject archive ls latest as "latest" is the name of a directory.
